----------
Instructions:
----------
Remember that hex code follows the red-green-blue, or rgb format. The first two digits of hex code represent the amount of red in the color. The third and fourth digit represent the amount of green. The fifth and sixth represent the amount of blue.So to get the absolute brightest green, you would just use F for the third and fourth digits (the highest possible value) and 0 for all the other digits (the lowest possible value).Make the body element's background color green by giving it the hex code value of #00FF00.

----------
Answer:
----------
<style>
  body {
    background-color: #00ff00;
  }
</style>
